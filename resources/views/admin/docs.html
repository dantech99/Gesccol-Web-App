Las secciones

las secciones son los diferentes bloques en que se divide el panel de registros del componente administrativo, para poder entender mejor como esta distribuido la aplicacion, hay que explicar que existen actualmente 6 tablas de registros, las cuales almacenan toda la informacion que se muestra en la aplicacion web, estas tablas son.

  • Categorias: que esta relacionada con los post, sirve para administrar las categorias que se crean
  • Noticas/Post para administrar los posts que se publican y van dirigidos al publico, tiene una relacion con las categorias para que estas puedan ser filtradas por las mismas
  • Servicios: para administrar el registro de todos los servicios que la empresa ofrece a la ciudadania
  • Contratos:
  • para administrar todas las entradas de contratacion que la empresa decide publicar para mostrar a los entes legales la legalidad de dichos contratos
  • Ordenes para administrar las ordenes que se llevan a cabo en la empresa, productos, vienes o servicios se publican aqui
  • Glosario: Para administrar todos los terminos desconocidos de catastro que se publican en la pagina de glosario de terminos

  • Como se crea una categoria

    Las cagorias estan relacionadas con las noticias, esto quiere decir que solo se pueden usar para clasificar una noticia en cuestion, ejm: si la noticia se publicara como comunicada, se puede crear una categoria con el nombre "comunicado", veamos como se hace.

    • dirigirse al formulario para crear una categoria en ðŸ‘‰ http://gesccol.gov.co/admin/categories/create al hacerlo aparece un formulario con varios campos en donde debera llenar los datos que se piden, el slug se llena solo. y crear cateogira, esta aparecera creada en el listado de categorias ðŸ‘‰ http://gesccol.gov.co/admin/categories la categoria se ha creado con exito, ahora vamos a asignarsela a una noticia.

  • Â¿Como se crea una noticia?.

    • las noticias o comunicados se crean de la misma forma que una categoria, se accede al formulario en la pagina ðŸ‘‰ http://gesccol.gov.co/admin/posts o por medio del panel de navegacion

      se abrira la pagina con el formulario para crear una noticia en el campo categoria, se elige entre todas las categorias creadas anterioremente, se llenan los demas campos y se opime publicar, esta noticia se vera reflejada en el listado de noticias y tambien en el componente usuario en la pagina ðŸ‘‰http://gesccol.gov.co/posts automaticamente.

  • Como se crea un post o noticia

    los post o noticias son publicaciones que se crean para informar al ciudadano aserca de nuevas novedades. En la barra de navegacion encontraran una opcion llamada crear noticia la cual redirecciona a un formulario con varios campos los cuales se deben llenar, campos como titulo, descripcion de la noticia e imagenes relacionadas a esta, asi al llenar dichos campos procedemos a pinchar el boton crear y este se creara automaticamente, pueden verificar que estÃ© creada corrctamente en la opcion, listado de noticias, este nuevo registro tambien se vera reflejado en la pagina de noticias y comunicados en el component usuario.

  • Como se crea un servicio.

    los servicios son los procesos que el usuario puede realizar en la empresa. Para crear un servicio se debe hacer el mismo procedimiento que se hizo con las noticias, solo que en este no existe el campo de las imagenes.

  • Como se crea una contratacion y una orden.

    Las contrataciones y Ordenes son registros que la empresa lleva para garantizar la transparencias de los recursos bienes y asociados. Para crear dichos registros se debe llenar los campos en el formulario, todos los campos son obligatorios, no se debe dejar ninguno vacios, asi que se tiene que verificar que el registro proporcionado por el directivo contenga los campos acordados.

  • como se crea-una palabra clave

    Las palabras claves no son mas que conceptos relacionados con catastro, para hacerlo se sigue el mismo procedimiento que en los demas registros

  • Como puedo cambiar la imagen de una noticia

    Para editar un registro ya creado se debe posicionar sobre la seccion de Noticias/post en la tabla en donde se listan todos los registros.

    Y oprimir el boton editar, esto lo llevara a un formulario en donde vera todos los campos del post en espesifico

    Como se aprecia los campos estan llenos, pero son el contenido que tiene el post en espesifico, asi que si quiere editar alguno solo debe borrar y cambiar lo que este mal o lo que necesite agregar. Para cambiar la imagene es muy sencillo, solo pinche en el boton chose File, se le abrirar el administrador de archivos para elejir su imagen y listo, procede a actualizar publicacion y si todo esta correcto el registro se actualizara, puede ir a verlo en la pagina ðŸ‘‰ https://gesccol.gov.co/posts
